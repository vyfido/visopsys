All of the files are zipped.  You will need a program such as WinZip (Windows) or GNU unzip (Linux/UNIX) to unzip them.  These are the different kinds of files available for download:

ISO image

Bootable .iso CD-ROM image file which includes the full Visopsys distribution.  You can burn the ISO file to a CD-ROM, or use it directly with an emulator such as VMware or Virtual PC.

USB image

Bootable .img image file which includes the full Visopsys distribution.  It can be written to a (minimum 30Mb) USB memory stick using a program such as Image Writer For Windows (Linux/UNIX users can use the built-in 'cat' or 'dd' commands).  This is a full-device image, including a partition layout.

Floppy disk image

Bootable .img file which includes a minimal Visopsys distribution.  It can be written to a 1.44 MB floppy disk using a program such as Image Writer For Windows (Linux/UNIX users can use the built-in 'cat' or 'dd' commands), or used with an emulator as above.

VMware image

A directory of files that can be used with the VMware emulator as a preinstalled Visopsys image.  You need VMware in order to use this.

Source Code

The files necessary to modify, build, and install the system.
