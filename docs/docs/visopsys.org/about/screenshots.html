SOME VISOPSYS SCREENSHOTS

Here are some images of Visopsys in action.  For these, I ran Visopsys under Linux using the VMWare Emulator.  Screenshots were then made using the GIMP. 

These images demonstrate the current console-based appearance of Visopsys, not the graphical one that is planned.  The commands executed in these screens are simple examples, using the primitive built-in command shell.  This shell, or SiSh, is a primitive user environment that Visopsys will use until the "real" user environment is available.
