You've reached Visopsys' home on the Web.   Visopsys is a new operating system for IBM-PC compatible computers.  Development is ongoing.  This kernel is small, fast, and open source.  It operates exclusively in 32-bit mode and features real preemptive multitasking and virtual memory.  This site is where you'll be able to download the latest version or get up-to-date information about the project.
