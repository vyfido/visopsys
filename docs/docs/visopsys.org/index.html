You've reached Visopsys' home on the Web.   Like Visopsys itself, this site is in a continuous state of development.  Visopsys is a new operating system kernel for IBM-PC compatible computers.  This kernel is small, fast, and open source.  It operates exclusively in 32-bit mode and features real preemptive multitasking and virtual memory.   Here is where you'll be able to download the latest version or get up-to-date information about the project.

The first public release of Visopsys (version 0.1) is now available to the public.  See the download page.

The fact that you're here probably means you're interested in operating systems development on the Web.  If you want to know more about Visopsys, please see the page about Visopsys.
