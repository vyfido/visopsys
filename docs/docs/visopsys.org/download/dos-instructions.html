HOW TO INSTALL VISOPSYS USING A DOS/WINDOWS SYSTEM

OVERVIEW

This file contains detailed instructions for installing Visopsys if you are currently running DOS/Windows. The files in this directory are the tools you will need for doing this.

INSTALLATION INSTRUCTIONS

Your top-level Visopsys directory should contain the following:

README.TXT  General README file.
dosutil This directory
instfile Contains the Visopsys files that get installed
unixutil Utilities for installing Visopsys using a UNIX system

If you are using Windows to install Visopsys, you should simply open a "Windows explorer" and click over to the directory where you un-zipped Visopsys. If you are using plain DOS, or would prefer to use a text interface, you should open a command prompt in the same Visopsys directory. In either case, you should now proceed to the directory called "dosutil". You should see the following files:

.                  <DIR> 04-01-00 4:17p  .
..                 <DIR> 04-01-00 4:17p  ..
FLOPPY~1 BAT      1,207 04-01-00 11:13p floppy-install.bat
WRITEB~1 COM      1,132 04-01-00 11:13p writeboot.com
INSTALL TXT       3,329 04-01-00 4:22p   INSTALL.TXT
        3 file(s) 5,668 bytes
        2 dir(s) 5,128.72 MB free

First, you need to insert a floppy disk in your A: drive. Note that the process will NOT work with a second (B:) floppy drive. Please use a floppy disk that is empty or conatins unwanted data, as this installation process will ERASE ALL DATA ON THE FLOPPY WITHOUT ANY FURTHER PROMPTING. Make sure the floppy disk is not write-protected (see the little movable tab on the back side of the floppy disk -- the tab should cover the hole).

The file you need to run to install Visopsys is floppy-install.bat (under Windows, you can simply double-click the file to install Visopsys, and you're finished). In a command prompt, type:

floppy~1 (or type the full name, floppy-install)

You do not need to type the .bat filename extension, but it will still work if you do so. You should see output similar to the following:

FLOPPY-INSTALL - Visopsys floppy disk installation script for DOS
Copyright (C) 2000 Andrew McLaughlin
-= NOTE: suitable for use only with Visopsys BINARY distribution =-

Formatting...
Copying os loader...
Copying kernel...
Installing boot sector...

WRITEBOOT - Visopsys boot sector utility for DOS
Copyright (C) 2000 Andrew McLaughlin
Writing A: boot sector with file: ..\instfile\bootsect.f12
done.

The entire process should take about 15 seconds. If you do not see any unusual error messages, then Visopsys is now installed on your floppy disk.  You should now be able to reboot your system using the Visopsys floppy you've just created. Tip: make sure your system BIOS is set to be bootable from a floppy. If you insert the floppy and reboot the system, but do not see messages about Visopsys, you need to enter your BIOS setup and change your "boot sequence" to something like A,C or A,CDROM,C (this is BIOS-specific).   The point is that the computer should first attempt to boot from the floppy drive where you insert the Visopsys boot diskette.

Good luck!
