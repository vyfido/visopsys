RELEASE 0.2

Floppy disk image:
visopsys-0.2-img.zip (680K)

Windows users:  Particularly if you are running Windows 2000, XP or greater, the Java installer in the binary distribution probably won't work for you.  Download the image file above and write it to a floppy disk using the rawritewin (266K) program, which should also work for Windows 95/98/ME.  Linux/UNIX users can also install the image file.  Unzip and 'cat' or 'dd' it to your floppy device.

Binary distribution:
visopsys-0.2-bin.zip (2960K)
Source code:
visopsys-0.2-src.zip (3148K)

Note that to use the installation program provided with the binary distribution, you will need to have Java on your system.  If you don't have it, you can get it from Sun Microsystems here, or use the image file download instead.
